# Familiarbre

## Install
### Java

Version 17+


## Run
### SpringBoot application

On port : 8080
```sh
cd jar
// Need Java 17+
java -jar familiarbre-0.0.1-SNAPSHOT.jar
```

### React application

On port : 5173
```sh
cd react/frontend
npm install
npm run dev
```

## Docs
To read the API docs go to : localhost:8080/http://localhost:8080/swagger-ui/index.html
